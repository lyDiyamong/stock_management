# Stock Management System

A Python-based inventory management application that provides comprehensive tools for managing product inventory, processing transactions, and generating analytics reports.

## Overview

This project implements a stock management system with modular architecture for handling data persistence, product operations, and business intelligence reporting.

## Features

- **Data Management**: JSON-based file I/O with automatic ID generation
- **Product Operations**: Full CRUD (Create, Read, Update, Delete) functionality with stock transaction tracking
- **Analytics & Reporting**: Generate comprehensive inventory reports and analytics

## Project Structure

- `data_manager.py` - Handles JSON file I/O operations and unique ID generation
- `product_manager.py` - Manages product CRUD operations and stock transactions
- `reports.py` - Generates inventory analytics and generates reports
- `main.py` - Entry point for the application

## Requirements

- Python 3.x

## Installation

1. Clone or download this repository
2. Ensure Python 3 is installed on your system

## Usage

Usage:
python3 main.py

Modules:
data_manager - JSON file I/O and ID generation
product_manager - CRUD and stock transaction operations
reports - Inventory analytics and report generation
